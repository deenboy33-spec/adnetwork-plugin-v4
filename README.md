# ADNetwork — Advertising Network Platform v4.0

**WordPress Plugin für Werbenetzwerk-Betreiber.**

## Features

### Modular System
Jedes Feature kann einzeln aktiviert/deaktiviert werden:

- ✅ **User System** (erforderlich) — Login, Register, Profile
- 🔄 **Campaigns** — Banner, Textlinks, Popups, MailAds
- 🔄 **Publisher** — Website-Registrierung, Embed-Codes
- 🔄 **Finance** — Guthaben, Transaktionen, Auszahlungen
- 🔄 **GSC-Coin** — Wallet, Vault, Shredder
- 🔄 **Referral** — 15-Ebenen-System
- 🔄 **Surfbar** — Auto-Surf
- 🔄 **Click4Win** — Klick-Spiel
- 🔄 **Lucky Wheel** — Glücksrad
- 🔄 **Search Game** — Suchspiel
- 🔄 **CoinFlip** — Münzwurf
- 🔄 **Dice** — Würfel
- 🔄 **Rallies** — Wettbewerbe
- 🔄 **Cashback** — Geld zurück
- 🔄 **Jackpot** — progressiver Jackpot
- 🔄 **PaidMails** — bezahlte E-Mails
- 🔄 **Walls** — ADCityCentral Walls + ABO
- 🔄 **ADS-MEDIA** — ADS-MEDIA Integration

## Installation

1. Plugin in `wp-content/plugins/adnetwork/` hochladen
2. WordPress Admin → Plugins → ADNetwork aktivieren
3. ADNetwork → Modules → gewünschte Features aktivieren
4. ADNetwork → Settings → Grundeinstellungen vornehmen

## Systemanforderungen

- WordPress 6.0+
- PHP 8.0+
- MySQL 5.7+ oder MariaDB 10.3+
